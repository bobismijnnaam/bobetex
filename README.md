Bobetex
=======
###### An engine for hybrid development on native and on the web

_Under development_

Make sure you have SDL2 installed. Assumed path is `/usr/include/SDL2` for headers. libSDL2.a is assumed to be in path.

Then:

```bash
mkdir build && cd build
cmake ..
make
```
